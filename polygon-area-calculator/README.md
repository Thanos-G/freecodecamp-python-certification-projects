# Polygon Area Calculator

Python project developed as part of the FreeCodeCamp Scientific Computing with Python certification.

## Description

The Polygon Area Calculator implements a simple geometry system using object-oriented programming.  
It defines classes representing geometric shapes and provides methods to calculate their properties.

The project includes a Rectangle class and a Square class that inherits from it, demonstrating basic inheritance and class design.

## Features

• Create rectangle and square objects  
• Calculate area, perimeter, and diagonal length  
• Generate a visual representation of the shape using text  
• Determine how many times one shape can fit inside another  

## Concepts Practiced

- Object-Oriented Programming (OOP)
- Class inheritance
- Method design
- Geometry calculations
- Python class structure
