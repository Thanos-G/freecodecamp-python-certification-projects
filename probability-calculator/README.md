# Probability Calculator

Python project developed as part of the FreeCodeCamp Scientific Computing with Python certification.

## Description

The Probability Calculator estimates probabilities using repeated random sampling.

A container of colored balls is simulated and experiments are performed to estimate the probability of drawing specific combinations.

## Features

• Simulates random ball draws from a container  
• Supports multiple colors and quantities  
• Performs repeated experiments  
• Estimates probability through Monte Carlo simulation  

## Example

Container:

- 5 blue balls
- 4 red balls
- 2 green balls

Experiment:

Draw 4 balls and calculate the probability of obtaining at least:

- 2 blue
- 1 red
