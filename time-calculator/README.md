# Time Calculator

Python project developed as part of the FreeCodeCamp Scientific Computing with Python certification.

## Description

The Time Calculator function adds a duration to a given start time and returns the resulting time and day.

The program supports:
- AM/PM time format
- optional starting day of the week
- time calculations that span multiple days

## Features

• Parses 12-hour clock input format  
• Calculates time across day boundaries  
• Handles optional weekday input  
• Returns formatted output including day and number of days later  

## Example

Input:

start = "3:30 PM"  
duration = "2:12"

Output:

5:42 PM

Example with weekday:

start = "11:55 AM, Monday"  
duration = "3:12"

Output:

3:07 PM, Monday
