# Budget App

Python project developed as part of the FreeCodeCamp Scientific Computing with Python certification.

## Description

The Budget App simulates a simple financial management system that tracks deposits, withdrawals, and transfers between budget categories.

It also generates a visual spending chart based on category expenses.

## Features

• Create multiple budget categories  
• Record deposits and withdrawals  
• Transfer funds between categories  
• Track account balances  
• Generate a spending chart showing percentage of expenses by category  

## Example

Categories:

- Food
- Entertainment
- Business

The program calculates the percentage of spending per category and displays a vertical bar chart.

## Concepts Practiced

- Object-Oriented Programming (OOP)
- Class design
- Data tracking
- String formatting
- Basic data visualization
