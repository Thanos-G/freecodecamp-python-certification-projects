# Arithmetic Formatter

Python project developed as part of the FreeCodeCamp Scientific Computing with Python certification.

## Description

The Arithmetic Formatter function arranges arithmetic problems vertically and optionally displays the calculated results.

The program validates input and formats the output according to standard arithmetic layout rules.

## Features

• Supports addition and subtraction problems  
• Validates input format and operators  
• Aligns numbers correctly for vertical display  
• Optionally displays calculated results

- String formatting
- Input validation
- Algorithmic thinking
- Python functions
